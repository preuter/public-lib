getBits()

getBits(  $n_bits) : \{int}

Extracts the requested number of bits from the 6-bit packed data. The extraction process is like a DFA (deterministic finite automata) that can't be accessed out of order. Once the data has been set, you can only march forward through the data. If you need to regrab an earlier portion of data, you'll have to reset().

Parameters

$n_bits

Returns

\{int} —

The value of the bits at the current state pointer.

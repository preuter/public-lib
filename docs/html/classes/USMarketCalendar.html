getOpen()

getOpen(integer  $ts = null, integer  $rel) : integer

Return the timestamp of a market open.

Parameters

integer $ts

Relative time to calculate market open from.
integer $rel

Indicates which market day to use. If N > 0: Get the Nth next market open. If N < 0: Get the Nth last market open. If N == 0: Get today's market open, or prev if not market day.

Returns

integer —

epoch timestamp of market open.

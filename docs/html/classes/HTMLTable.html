extractTableHTML()

extractTableHTML(  $src, string  $target, integer  $numShifts) : string

Extract table market from an HTML document starting from $target.

Search for $target and extract the table $numShifts after the text. If $numShifts is negative, counts backwards.

example: Extract table containing anchor: ($src,$target,-1) example: Extract very next table after anchor: ($src,$target,0) example: Skip next table, extract second: ($src,$target,1)

Parameters

$src
string $target

What to look for in html (case insensitive)
integer $numShifts

Number of tables to count.

Returns

string —

HTML containing the requested table.

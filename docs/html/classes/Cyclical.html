vectorize()

vectorize(float  $value, float  $mn, float  $mx) : \float[2]

Return an array of u, v components representing location on unit circle where $value is between $mn and $mx. Note, $mx should be 1 unit larger than the maximum possible value.

In a day, there are 1440 minutes. $mn = 0, $mx = 1440, since 1440 would be the next day.

Values larger than $mx or smaller than $mn will be wrapped. eg. Given: Value=3600, Range: [-180,180). Value wraps to (300-(180--180)) = -60.

Parameters

float $value

Number to convert from scalar to vector [u,v].
float $mn

Minimum value allowed in cycle range.
float $mx

Maximum value allowed in cycle range.

Returns

\float[2] —

array containing [u, v] vectorized cycling value.

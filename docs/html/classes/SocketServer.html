setShutdownTime()

setShutdownTime(integer  $ts) : boolean

Schedule service to shut down at time or after period of time.

When $ts is negative, the service will shut down abs($ts) sec from now. If positive, is assumed to be an epoch timestamp, when service should die. Otherwise, attempts to parse string with strtotime($ts)

Parameters

integer $ts

Either epoch timestamp or negative duration to live.

Returns

boolean —

true if time is valid, false otherwise.

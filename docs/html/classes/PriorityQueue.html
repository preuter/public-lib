_shiftDown()

_shiftDown(\uint  $pix) : \uint

Compare parent at $pix to children, sift down until valid queue.

Called on dequeue, which replaces first priority with last prior to sift. We must push this down the queue until priority strictly less than both children, since technically, this item was added after other children.

Parameters

\uint $pix

Parent index to compare against children.

Returns

\uint —

Index where parent rests in a stable heap.

init()

init()

Initialize a Scheduler object.

Used in child processes to clear active and future processes. Helpful to conserve memory and avoid shutdown confusion caused when there are no pending processes in the parent. The parent must issue a pcntl_waitpid($pid,$status) on all presently running processes to syncronize and signal proper termination.
